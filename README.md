# IMOVI - Projeto de Estudo com Bootstrap

Este projeto foi desenvolvido acompanhando uma aula no YouTube com o objetivo de entender melhor o funcionamento do **Bootstrap 5**.

## Objetivo

A intenção principal foi:

- Entender a estrutura do Bootstrap
- Trabalhar com componentes como Navbar e Carousel
- Compreender o funcionamento de classes utilitárias
- Praticar organização de HTML

## Contexto

Meu foco principal atualmente é:

1. Cibersegurança
2. Python
3. Fundamentos sólidos de desenvolvimento web

Decidi estudar Bootstrap por curiosidade técnica e para compreender melhor projetos legados ou sistemas que ainda utilizam esse framework.

## Tecnologias utilizadas

- HTML5
- Bootstrap 5
- Bootstrap Icons
- CSS personalizado

## Observação

É apenas um projeto de aprendizado para reforçar conceitos e explorar o funcionamento interno do Bootstrap.
